# Malay lesson 4 Colors, body parts

macam mana? (cam na?)

maca gitu lah (cam tu lah) = it's so so

akan = will (I will be going) saya akan jumpa doctor gigi

fever = demam kemarin mak saya demam (mom had a fever two days ago)

warna = color warna-warna = colors

white = putih black = hitam blue = biru merah = red jingga = orange hijau = green kuning = yellow ungu = purple coklat = brown kelabu = grey

kuching saya putih (my cat is white)

langit biru (sky is blue)

langit-langit = roof of your mouth

Saya tinggal di Bukit Merah (I live in Bukit Merah)

Minuman itu berwarna jingga ber (prefix) = ada (have) Minum-an = drink (noun)

da-un ini hijau = that leave is green Saya suka makan sayur hijau = I like eating green vegetables

pisang itu kuning = that banana is green

beli = buy saya nak beli pisang goreng = I want to buy fried bananas

Bunga itu berwarna ungu = That flower is purple

Kerusi itu berwarna coklat = that chair is brown

merah jambu = pink

Langit dah tukar jadi kelabu dah = sudah = already happened tukar = change

tukar duit = change money duit = money

pokok = tree brinjal = terung

Sayur terung berwarna ungu

awan = cloud Ada awan kelabu di langit = There are grey clouds in the sky

**Body parts = Anggota Badan**

kepala = head rambut = hair kening = eye brow bulu = generic body hair (apart from hair and eye brow) mata = eye hidung = nose te-ling-a = ear mulut = mouth bibir = lips (pron. bee-bay) gigi = teeth lidah = tongue pipi = cheek dagu = chin muka = face leher = neck (pron. ley-hey)

gatal = itchy small = kecil pain = sakit

Lidah saya sakit = My tongue is painful

Pipi saya tembam = my cheeks are chubby Dagu dia tajam = His chin is sharp

kacak = handsome cantik = pretty awak = you (more polite than kau) besah = big
