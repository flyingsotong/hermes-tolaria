**From Runway 02**

020 → 290 → 200 → 110 → 020

**From Runway 20**

200 → 110 → 020 → 290 → 200


| | | |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
|**ON THE RUNWAY** | | |
| Confirm correct runway | Referencing the Magnetic Compass or Heading Indicator | |
| Radio Calls | Rolling Runway radio call: _Traffic Murray Bridge,_ _Jabiru 1849,_ _Rolling Runway 02/20,_ _Murray Bridge Traffic_ | |
| Set to full power and maintain centreline | | |
| Callout | Runway Clear and Centreline Clear T’s and P’s are in the Green Airspeed alive Rotate at 65 kts | IF UNABLE TO MEET ANY OF THESE REQUIREMENTS, ABORT TAKEOFF IMMEDIATELY IF THERE ARE SUFFICIENT REMAINING RUNWAY |
| Rotate at 65 kts | Pitch the aircraft up to horizon | |
|**UPWIND** | | |
| Maintain Pitch | Dashboard on the horizon until 200 ft AGL (400 ft) | |
| Flaps Up | Callout: _200 AGL (400 ft),_ _Positive Rate of Climb_ _Flaps up_ | If unable to meet these requirements, monitor ﬂight instruments 200AGL is required to be cleared especially when taking off from runway 27 due to terrain on departing end of runway |
| Adjust pitch | Pitch for Vy 80 kts — best rate of climb Trim for climb attitude | |
| Turn to Crosswind | Turn to crosswind at 500 AGL (700 ft) | Lookout for other traffic prior to making turns (Verbal) |
|**CROSSWIND** | | |
| Level off wings | Level off on crosswind at 90 degrees — perpendicular from runway (Bearing 290, if departing Runway 02) | Verify 90 degrees by looking outside and referencing Magnetic Compass or heading indicator. MAINTAIN 90 DEGREES TRACK |
| Level off altitude | Level off at 1,000 AGL (1,200 ft), or turn to Downwind if already at 45 degrees from runway | MAINTAIN ALTITUDE THROUGHOUT CIRCUIT UNTIL TIME TO DESCEND |
| Turn to Downwind | Initiate turn to Downwind at 45 degrees from runway (Bearing 020 of departing Runway 20) | Lookout for other traﬃc prior to making turns (Verbal) |
|**DOWNWIND** | | |
| Level off when parallel to runway | APT**Attitude** level, hold at 1,200 ft**Power** cut to 23 MP**Trim** | MAINTAIN PARALLEL TRACK |
| Radio call | Downwind Radio Call _Traffic Murray Bridge_ _Jabiru 1849_ _Turning DOWNWIND Runway 20_ _Murray Bridge Traffic_ | |
| Checklist flow | 1. Fuel pump on + fuel selector 2. Carb heat + Brakes off 3. Seatbelts + Harnesses 4. Fuel to go around 5. Ts + Ps in green | |
| Power to descend power | PAT CARB HEAT ON Power down to 1,500 RPM at 45 degrees from the runway Pitch nose up to reduce speed if needed | |
| Turn to Base | Turn to Base at 45 degrees from runway Reduce speed to WHITE ARC for flaps | Lookout for other traffic prior to making turns (Verbal) |
|**BASE** | | |
| Flaps to 1 stage | Visual and Verbal call-out | Ensure that airspeed is in the white arc before putting down flaps or risk damage |
| Level off wings | Level off on base when 90 degrees/perpendicular to runway | MAINTAIN PERPENDICULAR TRACK |
| Radio call | Base Radio Call _Traffic Murray Bridge,_ _Jabiru 1849,_ _Turning BASE 02/20,_ _For Touch and Go / Full Stop,_ _Murray Bridge Traffic_ | |
| Adjust power + pitch for 70 kts | Adjust power as needed | Ensure a good descend profile |
| Turn to final at 10 o’clock from runway | Lookout for other traﬃc prior to making turns (Verbal) Take note of the winds and adjust bank angle to avoid overshooting or undershooting the runway | |
|**FINAL** | | |
| Level off wings | Level off when aligned with runway | MAINTAIN TRACK ALIGNED WITH RUNWAY |
| Adjust power + pitch for 65 kts | | |
| Verify runway is clear | Ensure runway is clear by looking and listening out — prepare for potential go-around | |
| Carb heat OFF | Carb heat is turned off in case go-around is required | |
