# Fitri, Alan Malay lesson 17 – September 28, 2022

Attendees: <fitriyanti@gmail.com>, Alan Soon

Minggu lepas kita belajar pasal apa? = Last week, what did we learn?

Lepas = already over lapar = hungry

berangin = got wind

bermakan angin = lit. eating air = traveling

dia tengah makan angin = he is in the middle of traveling (tengah is a continuous tense)

cerah = sunny/fair (and also be used to describe someone's fair skin) hujan = rain mendung / redup = overcast/gloomy

cheerful = ceria Pagi ceria = cheerful morning = good morning

I don't want to go out because it's raining = saya tak nak keluar rumah sebab hujan

gerimis = renyai-renyai = drizzle

tolong angkat kain baju. hari sudah gerimis. = please pick up the laundry, it's already drizzling

ribut petir (pron. pe-teh) = petir is lightning = ribut is thunder

ribut nak datang = a storm is coming

mendung = redup = overcast/gloomy

ber-awan = cloudy

minyak kapak = axe oil

minyak angin = "air medicated oil"

barang baik = good stuff

barang baik betul = good stuff for sure = bbb

salji = snow

i don't link snow because it is too cold = saya tak suka salji sebab telalu sejuk

kabus = fog saya tak boleh nampak sebab ada kabus
