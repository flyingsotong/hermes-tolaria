# Fitri, Alan Malay class – 5 October 2022 - Greetings revised

Attendees: <fitriyanti@gmail.com>, Alan Soon

tinggal = live = (also to say goodbye when leaving someone's house)

saya pergi dulu = I go first / I walk first (bye!)

balik dulu = go back (used at the end of the day)

selamat jalan = baik-baik jalan = be careful on your journey

baik = well baik-baik = be careful

bawa(k) kereta baik-baik = drive safely

tolong bawak-kan ni = please help carry/bring this

buka(k) puasa buka(k) pintu

datang = come datang lagi = come back soon

hati = heart

pengyu = turtle

saya tak permah makan teloh pengyu

one moment = sekejap / sebentar

happy weekend = selamat berhujung minggu (hujung = end)

Ahad = Sunday = Hari Minggu (end of the week)

selamat beristi rehat = rest well

macam mana = how is ______

how to do this? = macam mana nak buat ni?

nak = want = going to

I want to go to work = saya nak pergi kerja

makasih = thank you (abbreviated)

terima kasih banyak = thank you very much

ribuan terima kasih = thousand thanks

Saya minta maaf = Forgive me (minta is "ask")

Tah-niah = congrats Tak-ziah = condolences

saya nak pergi kedai, nak kirim apa? = I'm going to the shop, what do you need?

datang lagi = come back again

mari lah, jemput makan = come, let's eat

jemput = please

orang belum jemput, jangan minum dulu = you're not asked to drink yet, wait

directions = arah

go straight = jalan terus

teruskan = please continue

turn left = belok kiri (pron. bey-lohk) turn right = belok kanan

turn around = pusing belakang

make a u-turn = buat u-turn

look up = tengok atas = dongak look down = tengok bawah = tunduk

tengok-kan anak saya = please look after my child (-kan is a request) tengok anak saya = look at my child

panggilkan = call out on my behalf

masak-kan = cook for me masa-kan saya tak sedap = my cooking is not nice

we'll stop here = until here only = sampai di sini saja
