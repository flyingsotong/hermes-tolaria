**Greetings**


Selamat ten-gah hari = happy middle of the day = happy noon

Selamat petang (2pm - 7 pm) = good late afternoon

Selamat ber-isti rehat = rest well rehat = rest (saya nak rehat)

Selamat her-hu-jung ming-gu = Happy end of the week (ming-gu = week)

Selamat datang = welcome

Sihat = healthy Says sihat = I'm well (pron sey-HAT) Saya tak sihat = I'm not well Saya kurang sihat = I'm no so well

Awak macam mana = How are you Amacam = (short for macam mana) - MAT word

kurang = less kurang ajar = "less taught" = someone who's rude, not polite less sugar = kurang gula less salt = kurang garam less oil = kurang minyak

Selamat jalan = goodbye to guest Selamat ting-gal = goodbye to host (tinggal = stay)

Jumpa lagi = see you again Jumpa nanti = I'll see you at the next thing

Selamat sejahtera = Greetings (formal, used by broadcasters)

Saya minta maaf = Forgive me (minta is "ask")

Tah-niah = congrats Tak-ziah = condolences

Selamat hari jadi = happy birthday = happy the day you happened Selamat ulang tahun = happy repeat years = ulang is repeat

Sha-bas = well done Good = bagus or baik

Terima kasih untuk hari ini = thank you for today

Jumpa lagi minggu depan = see you again next week

masih hidup = still alive
