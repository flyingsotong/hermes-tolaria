# Cloud cover Clear, Few, Scattered, Broken, Overcast, Obscured

Clever foxes slyly baffle old owls.

| Category | METAR | Oktas (Eighths) | Description | Significance |
| --------- | ---------------------------------------------------------------------------------------------------------------------- | --------------- | ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| Clear | SKC (Sky Clear) | 0 | No clouds present. | Excellent visibility; minimal impact. |
| Few | FEW (followed by cloud base height, e.g., FEW025) | 1-2 | A small amount of clouds. | Generally good visibility; potential for isolated cloud effects. |
| Scattered | SCT (followed by cloud base height, e.g., SCT040) | 3-4 | More clouds, but still significant clear sky. | Visibility can be variably impacted; potential for changing conditions. |
| Broken | BKN (followed by cloud base height, e.g., BKN015) | 5-7 | Significant cloud cover, with breaks. | Potentially reduced visibility; risk of flying into clouds; variable solar radiation. |
| Overcast | OVC (followed by cloud base height, e.g., OVC008) | 8 | Complete cloud cover. | Reduced to poor visibility; potential for instrument meteorological conditions (IMC); significant impact on temperature. |
| Obscured | VV (Vertical Visibility, followed by distance, e.g., VV002), or other obscuration codes like FG (Fog), BR (Mist), etc. | N/A | Sky hidden by fog, precipitation, or other phenomenon. | Severely reduced visibility. Very dangerous for aviation. |

**Key Considerations:**
- Cloud base height is critical for aviation.
- Cloud cover influences temperature, precipitation, and solar radiation.
- Different cloud types also have different effects. Cumulonimbus clouds for instance, are very dangerous to aviation.

**Important Notes:**
- Vertical visibility (VV) is used when the sky is obscured, indicating how far an observer can see vertically.
- Obscuration codes such as FG, BR, and others, will be used in conjuction with cloud cover reports when applicable.
- Cloud heights are reported in hundreds of feet. For example, BKN015 means broken clouds at 1,500 feet.
