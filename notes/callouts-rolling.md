# Callouts Rolling

| | | |
| --------------------------------------- | ---------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Rolling | | |
| Prior to rolling runway checks/callouts | | This check/callout has to be done EVERY SINGLE TIME you're rolling on the runway. This includes rolling from a stop and a touch n go landing. |
| Runway clear, Centreline clear | Ensure runway is clear and able to maintain centreline | If runway is not clear, which could be caused by animals crossing or aircraft incursion, or, unable to maintain centreline such as winds, punctured tyres and etc, ABORT TAKEOFF IMMEDIATELY. |
| T's and P's are in the Green | Ensure that T's and P's are in the green and engine sounds right | If T's and P's are not in the green on full power, or aircraft does't sound right on full power, or engine RPM and Manifold Pressure doesn't look normal on full power, ABORT TAKEOFF IMMEDIATELY. |
| Airspeed is alive | Ensure that airspeed needle is increasing/moving | If airspeed is not increasing as the aircraft is speeding up, this could be caused by instrument/pitot static system failure. If this happens, ABORT TAKEOFF IMMEDIATELY. |
| Special Note | | In any case that a takeoff needs to be aborted, ensure power is DOWN TO IDLE, MAINTAIN DIRECTIONAL CONTROL USING THE RUDDERS, APPLY NECESSARY AMOUNT OF BRAKE FORCE and BRING THE AIRCRAFT TO A STOP OR EXIT THE RUNWAY IF OR WHEN IT IS SAFE TO DO SO. |
