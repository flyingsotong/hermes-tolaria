**Starting Up**
1. Brakes ON
2. Fuel tap ON and fuel content (Min 18 liters)
3. Hatches and Harnesses - Secured
4. Master ON
5. Radio OFF
6. Choke ON (if required)
7. Fuel Pump Test (6 sec)
8. Mags ON
9. Hand on throttle - "CLEAR PROP"
10. Start Engine
**After Start**
1. Check Oil Pressure (in the green)
2. Choke OFF
3. Avionics Master ON, Radio ON, Frequencies set (127.50 YMBD)
4. Radio Call "Radio Check" UHF/VHF: 
   _Traffic Murray Bridge,_
   _Jabiru 1849 requesting radio check,_
   _Murray Bridge Traffic_
5. Move to Run Up Position — Taxiing Radio Call: 
   _Traffic Murray Bridge,_
   _Jabiru 1849,_
   _Taxiing to hold point Alpha/Bravo,_
   _Murray Bridge Traffic_
**Pre-Take Off checks**
1. Brakes ON
2. QNH set (180 ft YMBD)
3. Rev up to 1,800 rpm
4. Mags check, 1 off/on, 2 off/on (max drop 100 rpm)
5. Carb Heat, on/off (observe slight drop in rpm)
6. Throttle back to IDLE (1,000 rpm)
7. CONTROLS free and correct (stick - forward, back left, right, around, check - elevator, ailerons)
8. FLAPS - one stage, two stage, set to one stage
9. TRIM Set for Take Off
10. Hatches and Harnesses secure
11. FUEL PUMP ON, LDG Lights ON
12. Radio call - Entering & Backtracking: 
    _Traffic Murray Bridge,_ 
    _Jabiru 1849,_ 
    _Entering and Backtracking Runway 02/20,_ 
    _Murray Bridge Traffic_
**Cruise checks**
2,600 rpm
Trim SET
**Downwind checks**
1. Fuel pump on + fuel selector
2. Carb heat + Brakes off
3. Seatbelts + Harnesses
4. Fuel to go around
5. Ts + Ps
**Approach Settings**
1. 65-70 kts
2. Flaps set as required
3. Trim BACK
4. Carb heat OFF (on short final)
**After landing checks**
1. Fuel pump OFF
2. Flaps UP
**Shut Down checks**
1. Brakes ON
2. Throttle down to 1,000 rpm, 60 sec min
3. Radio OFF, LDG Lights OFF
4. Throttle Release up, Close Throttle
5. Mags OFF
6. Master OFF
7. Maintenance Release paperwork complete (including fuel content remaining)
