We are entering a phase of endless media:**Created on demand, expansive, cheap**.

"We’re talking about a shift in how media is made and consumed. Rather than being created episodically, AI will enable it to be generated on demand." Mario Gabriele

<https://www.generalist.com/briefing/endless-media>

Where is value in a world of endless media?

- Content abundance reduces the value of institutional media -- human-curated services go to zero

- Uniquely "human" content will be elevated -- eg. emotional

Human IP vs Machine IP

**Human**

- Reddit -- increased value because it provides human insights and discussions

- Community -- value is in what specific people know that isn't already published

- Events -- human relationships and interactions

- Human-guaranteed content will be useful -- if it can be distinguished

- Made by a machine, guaranteed by a human
