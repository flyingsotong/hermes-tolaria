Minggu = week Se-ming-gu = one week

sepuluh = ten seratus = hundred (lit. 1-hundred = ser = 1)

wait a moment = tunggu sekejap one moment = sekejap / sebentar

Monday = isnin / senin Tuesday = selasa Wed = rabu Thu = khamis Fri = Juma-at Sat = Sabtu Sun = Ahad (or hari minggu)

Tomorrow = Esok / Besok (col.) day after tomorrow = Lusa

Yesterday = Semalam Last night = malam tadi Tonight = Malam ni

untuk = for

2 days ago = kelmarin (or ke-ma-rin for col.)
