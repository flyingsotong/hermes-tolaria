One more thing before you get into Thailand:

If you don't already have it, [make sure you download Grab](https://www.grab.com/download/) (the Uber of this region) before you arrive. Be sure to set it up with your credit card in advance so you can get around, especially from the airport! (You can also set it up for cash payments if that's easier.)
